# Sable Ledger

Welcome to the demo vault for **Sable Ledger**, an Obsidian theme shaped by warm editorial restraint.

## What to inspect first

- Sidebar contrast and file tree rhythm
- Tab treatment and active state subtlety
- Heading hierarchy in reading mode and Live Preview
- Inline code, code blocks, callouts, tables, and links

> [!note]
> This vault is the visual acceptance environment for the theme.

## Theme character

Sable Ledger is designed to feel like a quiet research desk rather than an app dashboard. Surfaces are paper-toned, typography is composed, and emphasis is used sparingly.

## Quick checklist

- Open this note in light mode
- Toggle to dark mode
- Visit `01-Longform Reading`
- Later, inspect the Dataview and Kanban notes

```md
The goal is not to mimic a website literally.
The goal is to make Obsidian feel authored.
```
