# Longform Reading

Good notes should feel readable before they feel technical. This page exists to check paragraph rhythm, hierarchy, block spacing, tables, quotations, and code under both light and dark appearance modes.

## Editorial pacing

Anthropic-inspired warmth works best when the interface stops fighting for attention. That means restrained contrast, gentle edges, and typography that gives ideas enough room to breathe.

### A blockquote

> Design should not merely disappear. It should support attention with enough character that the environment feels deliberate.

### A callout

> [!abstract]
> The theme should read like an intelligent publication system: quietly tactile, highly legible, and precise without becoming sterile.

## Mixed content

Inline code such as `app.workspace.getLeaf(true)` should be easy to scan inside prose. Lists should maintain shape even when they become nested:

1. Establish tone
2. Tune contrast
3. Refine interaction states
   - Hover should be visible
   - Focus should remain gentle
   - Active state should feel grounded
