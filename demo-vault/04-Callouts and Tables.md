# Callouts and Tables

This page stress-tests surfaces that often break visual consistency in note apps.

> [!tip]
> Callouts should feel like annotated paper slips, not loud dashboard banners.

> [!warning]
> The theme keeps warning surfaces calm, but they still need clear hierarchy.

## Nested information

> [!example]
> A callout can hold lists as well:
> - Editorial hierarchy
> - Controlled contrast
> - Quiet interaction

## Table comparison

| Pattern | Light mode target | Dark mode target |
| --- | --- | --- |
| Border | Warm grey | Soft parchment line |
| Header | Slightly raised | Slightly brighter surface |
| Code cell | Embedded | Embedded but readable |
